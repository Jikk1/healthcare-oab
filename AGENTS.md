# Инструкции для ИИ-агентов (Claude Code и др.)

Этот файл содержит правила и контекст для работы с проектом HealthCareOAB+. Любой агент, работающий в этом репозитории, **ОБЯЗАН** следовать этим инструкциям.

## 1. Режим глубокого планирования (Deep Planning Mode)

Перед внесением любых изменений в код агент должен войти в режим глубокого планирования:
1.  **Полная определенность:** Агент должен иметь абсолютную уверенность в ожиданиях и целях пользователя перед началом работы.
2.  **Подтверждение предположений:** Даже если задача кажется ясной, агент должен задавать уточняющие вопросы для проверки своих предположений.
3.  **Вопросы, а не утверждения:** На этапе планирования нужно задавать вопросы, а не предлагать готовый план на утверждение. План создается только ПОСЛЕ того, как все вопросы сняты.
4.  **Итеративность:** Агент должен задавать столько вопросов, сколько необходимо. Нулевое сомнение — критерий перехода к реализации.
5.  **Автономность после планирования:** Как только план утвержден, агент действует максимально автономно, не запрашивая подтверждения на каждом шагу, если это не предусмотрено планом.

## 2. Контекст проекта

*   **Суть:** Платформа предиктивной медицины нового поколения (Digital Twins, OmniRisk).
*   **Бэкенд (`backend/`):** Node.js, Fastify, Prisma, PostgreSQL. Основной фокус на безопасности PHI (данных пациентов).
*   **Фронтенд (корень):** Статический (сейчас) -> React (целевой).
*   **ИИ:** Движок `omnirisk.js` (клиент) и `risk-engine.ts` (сервер). Цель — переход на ML-модели (CatBoost/Python).

## 3. Правила разработки

1.  **Безопасность прежде всего:** Никогда не компрометируйте логику шифрования AES-256-GCM в бэкенде. Любое поле PHI должно быть зашифровано.
2.  **Многоарендность (Multi-tenancy):** Любой запрос к БД должен быть ограничен `organizationId`.
3.  **Документация:** Перед крупными изменениями сверяйтесь с `STRATEGY.md` и `INTEGRATION_PLAN.md`.

## 4. Как запустить Claude Code

Если вы используете `claude-code`, просто запустите его в корневой директории проекта:
```bash
claude
```
Он автоматически подхватит эти инструкции из `AGENTS.md`.
